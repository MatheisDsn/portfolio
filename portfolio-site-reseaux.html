PORTFOLIO

Page de destination pour les médias sociaux.

J'ai réalisé ce projet pour mon propre usage, afin de ne pas utiliser les outils disponibles sur internet.


L'objectif ? Simplifier l'accès à TOUS" les réseaux sociaux très simplement, afin de gagner en visibilité.

Ce projet simple mais fonctionnel m'a permis de personaliser en fonction de mes envies, ce qui n'était pas vraiment possible avec les outils prédisponibles.

Ici, l'image de fond change en fonction de certains Évènements.


Technologies utilisées : HTML, CSS
